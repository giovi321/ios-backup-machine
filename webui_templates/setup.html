2 WiFi (optional)

3 Date & Time

Set the system clock or enable automatic NTP synchronization. NTP requires an internet connection (WiFi or iPhone USB hotspot).

Only applied if you change this field. Leave as-is to keep the current system time.

4 Backup Storage

Where should iPhone backups be saved?

Default: /media/iosbackup/ - make sure the microSD card is mounted here.

5 Backup Encryption

iOS backups should be encrypted to protect your data. Choose a password that will be set directly on your iPhone. This password is NEVER stored on this device - write it down and keep it safe. You will need it to restore backups.
